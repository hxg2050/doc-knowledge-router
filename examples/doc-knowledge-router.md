# Doc Knowledge Router Examples

## Create a Docs Entry Point

```text
Use $doc-knowledge-router to create a docs/index.md entry point for this project. Keep it focused on where to find information, not on rewriting the actual system docs.
```

## Build Workflow Navigation

```text
使用 $doc-knowledge-router 整理 docs/workflows/index.md，让读者能根据“我要做什么”找到应该先读的文档。
```

## Resolve Duplicate Documentation

```text
Use $doc-knowledge-router to inspect duplicate docs, choose one authority location, and convert the other copies into short summaries with links.
```

## Archive Old Material

```text
使用 $doc-knowledge-router 把历史计划和过期审计材料归档，同时保证当前入口还能指向正确位置。
```
